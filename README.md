# Speech-Enhancement
基于深度学习的语音增强模型旨在通过从噪声语音中估计出清晰的语音信号来提高语音的质量以及可懂度。
在现实环境中，单个麦克风接收到的语音与噪声信号在时域上的关系表示为：

$$
y_t = s_t * h_t + n_t
$$

其中：
- $s_t$ 表示为纯净语音信号 
- $h_t$ 表示为房间脉冲响应 
- $n_t$ 表示为加性噪声 
- $y_t$ 表示为噪声信号

## 训练方法
* #### 1. Mapping 
	基于映射(Mapping)的方法，其核心在于将带噪语音直接转换为干净语音，这一过程可被视为一个回归问题，其目标是学习一个函数， 该函数能够最小化输入和期望输出之间的差异。
* #### 2. Masking
	基于掩蔽(Masking)的方法，其不直接“生成”干净语音，而是通过模型**估计一个“掩蔽函数”（Mask）**，用于从带噪语音的时频表示中**选择性地保留语音成分、抑制噪声成分**。
	估计掩蔽矩阵的问题可被视为一个分类任务。在这种方法中，输出的掩蔽函数可被看作一个实现对抑制噪声和增强语音信号的选择性滤波器。在具体实现上，这种方法通常涉及到对噪声和语音的统计特性进行建模，并利用这些模型来指导掩蔽矩阵的估计过程。

## 关于两种训练方法的进一步说明

### 基于映射的方法(Mapping) ###
为方便理解，我们不考虑脉冲信号对纯净语音信号的影响。将语音增强任务简单建模为：

$$
y(t) = x(t)+n(t)
$$

其中：
- y(t): 观测到的带噪语音（Noisy speech）
- x(t): 真实的干净语音（Clean speech）
- n(t): 背景噪声

我们的任务是从 $y(t)$ 中估计出 $\hat{x(t)}$ ，并使 $\hat{x(t)}$ 尽可能的等于 ${x(t)}$ 。对于映射的方法来说，训练一个模型 $f$ （如 DNN、CNN、Transformer 等），使得 $\hat{x(t)} = f(y(t))$ 或 $\hat{X(f,t)} = f(Y(f,t))$ 。即模型可以直接从输入的带噪语音（时域波形或频域谱图）映射到输出（增强后的语音）。

### 映射方式的分类 ###
**1、时域映射（Time-domain Mapping）**
- **输入**：带噪语音y(t)（时域信号）
- **输出**：估计的干净语音 $\hat{x}(t)$ （时域信号）
- **模型示例**：Demucs, Wave-U-Net, Conv-TasNet等等

**2、频域映射（Frequency-domain Mapping）**

**(a)幅度谱映射（Magnitude Spectrum Mapping**
- **输入**：带噪语音的 STFT 幅度谱 $|Y(f,t)|$ 
- **输出**：估计的干净语音的幅度谱 $|\hat{X}(f,t)|$
- **相位选择**：使用带噪语音的相位 $\angle Y(f,t)$

$$
\hat{x}(t) = \text{ISTFT}\left( |\hat{X}(f,t)| \cdot e^{j \angle Y(f,t)} \right)
$$
  
- **缺点**：忽略了相位的重要性，重建语音存在“机械声”或失真。

 **(b) 复数谱映射（Complex Spectrum Mapping）**
- **输入**：带噪语音的复数谱 $Y(f,t) = |Y| e^{j \phi_Y}$
- **输出**：干净语音的复数谱估计 $\hat{X}(f,t)$
- **相位选择**：对幅度和相位的同时建模：
 
$$
\hat{X}(f,t) = f(Y(f,t)) \in \mathbb{C}
$$
  
- **优点**：同时优化幅度和相位，显著提升语音质量和可懂度，且更符合真实语音生成过程。


### 基于映射模型的训练方式 ###
**1、监督学习（Supervised Learning）**
- 使用成对数据： $(Y, X)$ 学习，使得 $f(Y) \approx X$
- 损失函数：
    - **均方误差（MSE）**： $\frac{1}{n}{\sum_{i=1}^{n} (\hat{x_i}-x_i)^2}$
    - **频谱损失**：STFT 域的 L1/L2 损失
    - **感知损失**：SI-SNR、STOI、PESQ 等指标的可微近似

 **2、无监督/半监督映射**
- 使用未配对数据，结合 GAN 或自编码器结构。例如：GAN 中的生成器 GG 就是一个从噪声分布到干净语音的映射。


### 映射模型的优缺点 ###
**优点**：
- 概念简单，易于实现。
- 可支持端到端训练，性能强大。
- 支持时域、频域、复数域等多种表示。

 **缺点**：
- 依赖大量配对训练数据。
- 泛化能力受限于训练噪声类型。
- 可能引入人工噪声（over-smoothing 或 artifacts）。

### 基于掩蔽的方法(Masking) ###

**数学表达**
语音信号通常先转换到**频域**（如通过短时傅里叶变换 STFT）：
	$Y(f,t)=X(f,t)+N(f,t)$
其中：
- Y(f,t): 带噪语音的复数谱
- X(f,t): 干净语音的复数谱
- N(f,t): 噪声的复数谱
目标是估计一个掩码M(f,t)，使得：
	$\hat{X(f,t)}=M(f,t)⋅Y(f,t)$
这个 M(f,t)就是模型输出，它决定了每个时频单元上保留多少原始信号
